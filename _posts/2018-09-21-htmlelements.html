css positioning

Metadata Content

base Determines the base url for all links. When a link goes off to an external site this element has to appear before it.
link Used to connect a document to an external resource such as external stylesheets.
meta Specifies all the metadata within the html document, like author, description or keywords. This information is not visible on the page.
noscript If there are disabled scripts or issues with browser support noscript allows for alternate content. It can be used in the body and the head. If in the head elements are limited to link style and meta.
script Where scripts instructions are kept or link to external ones. Used to dynamically create block and/or inline content
style Defines style info in the document and specifies how different elements should be styled.
title title of a document, specified in the head section in an html document. The content entered here appears in the browser toolbar, creates a title when the page is added to favorites and displays a title in search engine results.
